  1. 什么是一级标注、二级标注、三级标注

    在微信读书中，标注分三类，分别是波浪线、马克笔和直线（如下图）：

    被直线标注的文本就是设置页面中所谓一级标注，相应地，被马克笔标注的文本是二级标注，被波浪线标注的文本是三级标注。

  2. 什么是前缀、后缀

    插件可分别为不同级别的微信读书标注补充文本，补充在标注前面的部分为前缀，补充在后面的部分为后缀。

    你可以在输入框中设置标注前后缀。设置为前缀的内容将会在导出文本的时候添加到标注开头，设置为后缀的内容则将会在导出文本的时候添加到标注之后。

    比如，如果你希望书本中二级标注（用马克笔标注的文字）导出来后在 Markdown 中加粗，你只需要将二级标注前后缀设置为两个星号：**

    又比如，如果你希望书本中用直线标注的文字（一级标注）导出来后在 Markdown 中加下划线，你只需要将一级标注前后缀分别设置为<u>和</u>。

    再比如，当你希望被二级标注（用马克笔标注的文字）导出来之后在 Markdown 中显示为无序列表。你只需要将二级标注后缀设置为空（也就是不设置任何内容），二级标注的前缀设置为一个短横杆加一个空格：-

  3. 代码块前缀、后缀

    当你借助插件复制书本中的代码块的时候，你可以在代码块前缀、后缀选项设置代码块前后缀。

    比如假设书中有一段代码，当你将代码块前缀设置为```python同时将代码块后缀设置为```后，你复制得到的文本在 Markdown中会按照 python 的语法高亮显示。

    有时你可能不希望导出的代码块前后包含任何内容，这时你可以将代码块前后缀设置为空。

  4. 正则表达式

    当微信读书中的三种标注不能满足你添加前后缀的需求时，你可以借助正则表达式匹配标注，被匹配的标注可被添加前后缀。

    比如，当你要匹配一个开头为"提示"二字的标注时，可输入正则表达式：

    ^提示[\S ]*$

    下面是一些比较常用的正则表达式：

    • 匹配开头为"步骤x："(x为0到99中任意整数)的标注：^步骤[0-9]{1,2}[:：][\S ]*$
    • 匹配图片标题如"图2-5 404错误响应"：^图[0-9]{1,2}-[0-9]{1,2} [\S ]*$
    • 匹配表格标题如"表1-1 Flask的依赖包"：^表[0-9]{1,2}-[0-9]{1,2} [\S ]*$
    • 匹配开头为数字加一点的标注如"1．Flask与HTTP"：^[1-9][．\.][\S ]*$
    • 匹配开头为"注意："或"附注："或"提示："的标注如"提示：你可以借助微信读书笔记助手快速导出笔记为 Markdown 文本"：^(注意|附注|提示)[:：][\S ]*$

    注意：

    • 选中正则表达式右方的选中框该正则表达式才会生效。
    • 因为正则表达式的设置很容易影响导出效果，所以最好在对正则表达式进行充分测试后再使用正则匹配。这是一个可供学习正则表达式的教程：learn-regex-zh
  5. 什么是一级标题前缀、二级标题前缀、三级标题前缀

    你平时在读书页看见的目录对应于不同级别的标题。

    你可以在设置页中设置标题导出级别。

    默认级别分别是二级：##、三级：###和四级：####。你可以通过自己改变井号数量来改变标题级别，从而得到不同的标题导出效果。

    事实上，修改标题级别本质上是在修改标题的前缀。

    注意：你需要在最后一个井号之后加一个空格该标注在 Markdown 中才会被解析为标题。

  6. 什么是想法前后缀

    想法也就是你平时读书的时候在书中发表的想法。前后缀与前面提到概念的一样。当你将前后缀设置为两个星号（**）时，导出的想法在 Markdown 中将会被加粗。

  7. 配置选项

    你可以在设置页中新建、删除、重命名以及切换配置选项。

    之所以提供建立多个配置选项的功能是因为不同的书往往需要不同的导出设置，为此，你可以为不同的书建立不同的配置，然后在需要使用该配置的时候选中即可。

  8. 其他
    • 由选项开启转义，你可以设置是否转义标注及纯文本书评中的 Markdown 特殊字符（如 \ * { [ ( + < _ ` !）。
    • 由选项导出标注包含想法，你可以设置导出的标注（本章标注或是全书标注）中是否包含你在该书中留下想法。
    • 由选项热门标注显示标注人数，你可以设置导出的热门标注中是否包含每条热门标注被标记的人数。
